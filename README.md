# Scientific-Calculator using Python - Tkinter
## Minimum specifications 
Python to be installed
<br>
Tkinter library to be installed
<br>
If the library is not installed run the following command on cmd to install it
<br>
### pip install Tkinter
## To run the appilcation
downlaod the above file and execute it(via python interpretor)
## About the project
A scientific calculator done using Tkinter module for UI
The working of Calculator - 1st the infix expression is converted to postfix expression then the postfix expression is evaluated
(calculate function is used to separate and differentiate operators and operands)
